STEPS TO RUN THE PROJECT

Open the movie-recom-system.ipynb file in Jupyter Notebook or Google Colab.
Run all the cells in sequence.
After executing all the cells, the required .pkl files will be generated.
Move or copy all the generated .pkl files to the same directory where main.py is located (or your preferred Python IDE/project folder).
Open the main.py file in your preferred Python IDE (VS Code, PyCharm, etc.).
Install the required dependencies (if not already installed).
Run main.py to start the Movie Recommendation System.
